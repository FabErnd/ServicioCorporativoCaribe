# ServicioCorporativoCaribe
Requisitos:
.NET SDK 6.0 o superior
SQLite

# Pasos para instalación:
# 1
Crear un nuevo proyecto usando 
dotnet new console -o NominaCaribe

# 2
Descargar todos los archivos del GitHub y arrastrarlos dentro de la carpeta del proyecto creado.

Ya con esto el programa debería ser enteramente funcional.

# Notas
Si quiere una base de datos limpia, no descargue o borre si lo hizo el archivo llamado nomina_caribe.db.
